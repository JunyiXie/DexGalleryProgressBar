# DexGalleryProgressBar
A ProgressBar For Gallery ...

**Demo**
![static](https://github.com/JunyiXie/DexGalleryProgressBar/raw/master/readme_resources/demo.gif)

**特征：**
1. CoreAnimation 动画
2. 滑动时显示进度条/时间块
3. 滑动条&时间块 联动处理
4. 大量照片情况下 滑动条长度处理
5. 大量照片情况下 滑动条 n次滑动照片/一次进度条移动处理
6. 滑块&进度条 边界情况处理
